# NeuroTrade V7 - Centro de Comando

Sistema de trading de alta frecuencia (HFT) impulsado por un ejército de IA con arquitectura multi-bróker.

## Despliegue en GitHub

Para subir este proyecto a tu repositorio, ejecuta los siguientes comandos en tu terminal:

```bash
git init
git add .
git commit -m "first commit: NeuroTrade V7 Core"
git branch -M main
git remote add origin https://github.com/vallrack/NeuroTrade.git
git push -u origin main
```

## Características V7
- **Núcleo Maestro:** Consenso de 5 agentes de IA.
- **Protocolos HFT:** Operativa en temporalidades de 1 minuto.
- **Umbrales RSI Cuánticos:** Configuración maestra en niveles 20, 38 y 62.
- **Arquitectura de Puente:** Soporte para IQ Option, Alpaca y Cripto (CCXT).
- **Control de Riesgo Blindado:** Gestión de Take Profit, Stop Loss y Kill-Switch global.
